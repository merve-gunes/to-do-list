# Vanilla JS To-Do List

Herhangi bir kütüphane veya framework (React, Vue vb.) kullanılmadan, tamamen **Saf (Vanilla) JavaScript** ile geliştirilmiş, verileri tarayıcı hafızasında saklayan bir Yapılacaklar Listesi uygulaması.

## Proje Hakkında

Bu proje, JavaScript'in temellerini, **DOM Manipülasyonunu** ve **LocalStorage** kullanımını pekiştirmek amacıyla yapılmıştır. Kullanıcı sayfayı yenilese veya tarayıcıyı kapatsa bile eklediği görevler kaybolmaz.

##  Özellikler

* **Görev Ekleme:** Kullanıcı metin girerek listeye yeni eleman ekleyebilir.
* **Veri Kalıcılığı (LocalStorage):** Sayfa yenilendiğinde veriler kaybolmaz, tarayıcı hafızasından geri yüklenir.
* **Tamamlandı İşaretleme:** Görevin üzerine tıklandığında üstü çizilir (Toggle mantığı).
* **Görev Silme:** Her görevin yanındaki buton ile o görev listeden ve hafızadan kalıcı olarak silinir.
* **Klavye Desteği:** "Ekle" butonuna basmadan, `Enter` tuşu ile hızlıca ekleme yapılabilir.
* **Boş Veri Kontrolü:** Boş bir görev girilmeye çalışıldığında kullanıcı uyarılır.
* **Dinamik DOM:** HTML elementleri JavaScript ile dinamik olarak oluşturulur.
